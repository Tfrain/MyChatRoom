## 启动
### 1:启动服务器
- 根据实际情况修改 *resources/server.properties* 下的配置文件
- 运行文件 *main/ServerStart.java*
### 2:启动客户端
- 根据实际情况修改 *resources/client.properties* 下的配置文件
- 运行文件 *main/ClientStart.java* (运行多次则启动多个客户端)
